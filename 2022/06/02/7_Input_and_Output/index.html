#7 输入与输出

2022-06-02
  • C++

输入输出流

C++的输入输出设施由iostream库提供，它是一个面向对象类层次结构，作为C++标准库的一个组件，为内置数据类型的输入输出提供支持，通过扩展iostream库来读写新的类类型，支持对终端、文件、内存的输入输出。

输入输出操作由如下对象来完成：istream输入流 ，ostream输出流 ，iostream（同时从istream和ostream派生） 输入输出流。

标准库定义了下列三个标准流对象：cin代表标准输入的istream对象， 从用户终端读入数据；cout代表标准输出的``ostream对象， 向用户终端写数据；cerr代表标准错误的ostream`对象，一般也是终端。

下列三种类类型提供了对文件读写支持：ifstream从istream派生， 绑定一个文件用来输入；ofstream从ostream派生， 绑定一个文件用来输出；fstream从iostream派生， 绑定一个文件绑用来输入和输出。

内存输入输出 ：当流被绑定在一个字符串上时，可用iostream的输入和输出操作符来对它进行读写：istringstream从istream派生， 从一个字符串中读取；ostringstream从ostream派生， 写入到字符串中；stringstream从iostream派生， 从读取或者写入到字符串。

输出操作符 <<

输出流的运算符：

cout << "Hello world!\n";

输出操作符参数可包括任何内置数据类型的实参， 包括const char*以及标准库string和complex类类型，任何表达式以及任何函数调用，只要其结果是一个能被输出操作符接受的数据类型。

如： 

cout << "The length of "ulysses" is:\t";
cout << strlen( "ulysses" );
cout << '\n';
cout << endl;
//endl是一个ostream操纵符（manipulator）换行。

输出操作符可连接成一条语句：

cout  << "The length of "ulysses" is:\t" << strlen( "ulysses" ) << '\n' <<endl;

对于自定义类型的输出，参见操作符重载。

格式化输出

//杨辉三角
#include <iostream>
using namespace std;
int fun(int i, int j) {
    if (j == 0 || j == i)
        return 1;
    else
        return fun(i - 1, j - 1) + fun(i - 1, j);
}

int main() {
    int n = 10;
    for (int i = 0; i < n; i++) {
        for (int k = 0; k < n - i - 1; k++)
            cout << " ";  //输出空格
        for (int j = 0; j <= i; j++)
            cout << fun(i, j) << " ";
        cout << endl;
    }
    return 0;
}

#include <iostream>
using namespace std;

int main() {
    bool flag = true;
    cout << "Default : " << flag << endl;
    cout << "Boolalpha : " << boolalpha << flag << endl;//boolalpha使得bool值输出true/false
    cout << "Noboolalpha : " << noboolalpha << flag << endl;//noboolalpha使得bool值输出1/0
    return 0;
}
//结果为：
//Default : 1
//Boolalpha : true
//Noboolalpha : 1

更多的格式输出控制在头文件 <iomanip>。

#include <iostream>
#include <iomanip>
using namespace std;
int main() {
    int a = 100;
    double pi = 22.0 / 7.0;
    const char *pt = "China";
    cout << "dec:" << dec << a << endl;  //以十进制形式输出整数
    cout << "hex:" << hex << a << endl;  //以十六进制形式输出整数a
    cout << "oct:" << setbase(8) << a << endl;  //以八进制形式输出整数a
    cout << setw(10) << pt << endl;  //指定域宽为10,输出字符串
    cout << setfill('*') << setw(10) << pt << endl;  //指定域宽,输出字符串,空白处以'*'填充
    cout << "pi=" << pi << endl;  //输出pi值
    cout << setiosflags(ios::scientific) << setprecision(8);//按指数形式输出,8位小数
    cout << "pi=" << setprecision(4) << pi << endl;  //改为4位小数
    cout << "pi=" << setiosflags(ios::fixed) << pi << endl;  //改为小数形式输出
    return 0;
}

输入操作符 <<

例子：从标准输入读入int型的值序列，并放在一个vector容器中：

vector<int> ivec;
int ival;
while ( cin >> ival )
//从标准输入读入一个整数值,如果成功,则把该值拷贝到ival中,这个子表达式的结果是左边的 istream 对象。
    ivec.push_back( ival );

两种情况会使一个istream对象被计算为false：读到文件结尾，此时已正确地读完文件中所有的值；遇到一个无效的值，比如3.14或其他非整数的字符串。

其他输入输出操作

istream

get()：一次读入一个字节。

重载1：get(char& ch)，从输入流中提取一个字符，并存储在ch中，返回istream对象。

char ch;
while ( cin.get( ch ))
    cout.put( ch );

重载2：get()从输入流读入一个字符，但返回该字符值。

int ch;
while (( ch = cin.get()) != EOF )//EOF:end of file
    cout.put(ch);

istream的其他成员：ignore，getline。

ostream

包括成员put()和write（）。

put()将字符输出到输出流中，参数为char，返回被调用的ostream类对象。

文件的输入和输出

<fstream>中的fstream类可对文件进行输入输出操作。

fstream的成员函数：

void  open(const char *filename,int mode,int access);
//参数：  
//filename：　　  //打开的文件名  
//mode：　　　    //打开文件的方式  
//ios::app：　   //以追加的方式打开文件 
//ios::ate：　　 //文件打开后定位到文件尾，ios:app就包含有此属性
//ios::binary： //以二进制方式打开文件，缺省的方式是文本方式。
//ios::in：　　  //文件以输入方式打开（文件数据输入到内存） 
//ios::out：　　 //文件以输出方式打开（内存数据输出到文件）
//ios::nocreate：//不建立文件，所以文件不存在时打开失败 
//ios::noreplace：//不覆盖文件，所以打开文件时如果文件存在失败 
//ios::trunc：　  //如果文件存在，把文件长度设为0
//打开文件的方式在ios类(所有流式I/O类的基类)中定义
//可以用“或”把以上属性连接起来，如ios::out|ios::binary
//access：打开文件的属性  
//0：普通文件  
//1：只读文件  
//2：隐含文件  
//4：系统文件 
//可以用“或”或者“+”把以上属性连接起来，如3或1|2就是以只读和隐含属性打开文件。  

例如：以二进制输入方式打开文件c:\config.sys：

fstream file1;
file1.open("c:\\config.sys", ios::binary | ios::in, 0);

若只有文件名一个参数，则是以读/写普通文件打开，即：

file1.open("c:\\config.sys");
//等价于
file1.open("c:\\config.sys", ios::in | ios::out, 0);

fstream有和open()函数参数一样的构造函数重载，对于上例，在定义的时侯就可以打开文件：

fstream file1("c:\\config.sys");

fstream的两个子类中，ifstream以输入方式打开文件，而ofstream以输出方式打开文件。

因此：以输入方式打开，用ifstream来定义；以输出方式打开，用ofstream来定义；以输入/输出方式来打开，则用fstream来定义。

打开的文件使用完成后要使用函数close()关闭。

下面程序读取了一个文件，并将内容写到标准输出上：

#include <fstream>

int main() {
    cout << "filename: ";
    string file_name;
    cin >> file_name;
    ifstream inFile( file_name.c_str() ); // 打开文件 copy.out 用于输入
    if ( !inFile ) {
        cerr << "unable to open input file: "  << file_name << " -- bailing out!\n";
        return -1;
    }
    char ch;
    while ( inFile.get( ch ))
        cout.put( ch );
}

文件的读写

读取一个文件的内容，写入另一个文件：

ifstream fr;
ofstream fw;
char word[200], line[200];
fw.open("write.txt");
fr.open("read.txt");
fr  >> word;    //读取文件，一个单词
fr.getline (line, 100);        //读取一行内容
fw << "write file test" << endl;      // 写入
fw.close();
fr.close();
return 0;

C++I/O系统管理两个与一个文件相联系的指针：一个是读指针，它说明输入操作在文件中的位置；另一个是写指针，它下次写操作的位置。每次执行输入或输出时，相应的指针自动变化。

ofstream的put_point简称p，即写指针，ifstream的get_point简称g，即读指针，fstream的put_point和get_point，简称同为p和g。

获取流指针位置：tellg()：返回输入流指针的位置（返回类型long），tellp()：返回输出流指针的位置（返回类型long）。

设置指针位置：seekg(long position)：设置输入流指针位置，seekp(long position)：设置输出流指针位置。

如seekg函数对输入文件定位：第一个参数是偏移量，正数表示向后偏移，负数表示向前偏移。第二个参数是基地址，表示输入流的某个起始位置，ios：：beg为开始位置，ios：：cur为当前位置，ios：：end为结束位置。

判断文件状态

is_open()：文件是否正常打开。

bad()：读写过程中是否出错（如操作对象是否打开，写入的设备有没有空间）

fail()：读写过程中是否出错（操作对象是否打开，写入的设备有没有空间，格式错误如读入类型不匹配等）

eof()：读文件到达文件末尾则返回true，否则返回false。
